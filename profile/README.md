# WUSLA

Independent product lab from Kerala, India.  
We design and build modern web and mobile platforms for startups and small businesses.

We focus on:

- Custom web apps with Next.js and the MERN stack
- Native‑feeling mobile apps with React Native
- Clear, minimal UI/UX and reliable engineering

Our goal is simple: ship practical tools that help people manage money, events, communities and everyday workflows.

## Products

- Finance Fund – Simple tools for managing personal and small‑business finances.
- Islamic AI – Context‑aware tools built around Islamic knowledge and workflows.
- Aether – Notes and tasks with a calm, spatial interface.
- WUSLA Web – Modern websites and landing pages for coaching centres, restaurants and service businesses.

## Tech Stack

- Frontend: Next.js, React, Tailwind CSS
- Mobile: React Native
- Backend: Node.js, Express, MongoDB, PostgreSQL, Supabase
- Infra & DevOps: Vercel, Docker, GitHub Actions
- Design: Figma
